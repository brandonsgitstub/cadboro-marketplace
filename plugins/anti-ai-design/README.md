# anti-ai-design

Distinctive, non-generic frontend interfaces.

Part of the [Cadboro Labs](https://cadborolabs.com) Claude Skills Marketplace.

## Install

```bash
/plugin install anti-ai-design@cadboro-labs
```

## Skill

- `frontend-design` — invoke with `/anti-ai-design:frontend-design`, or just describe the task and Claude triggers it automatically.
