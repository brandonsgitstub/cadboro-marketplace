# jtbd-research

Jobs-to-be-Done interview guides + synthesis.

Part of the [Cadboro Labs](https://cadborolabs.com) Claude Skills Marketplace.

## Install

```bash
/plugin install jtbd-research@cadboro-labs
```

## Skill

- `jtbd-interview-guide` — invoke with `/jtbd-research:jtbd-interview-guide`, or just describe the task and Claude triggers it automatically.
